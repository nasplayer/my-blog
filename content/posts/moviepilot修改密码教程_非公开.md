# Moviepilot修改密码教程

如何修改默认密码并关联上下载器和播放器呢？



# 1  修改Moviepilot密码

1.1 用《Moviepilot使用手册》里提供的MP链接 和 账号密码登录上MP

![image-20260301172352332](/images/moviepilot修改密码教程_非公开/image-20260301172352332.png)



 在点击上图1和2的位置后会弹出编辑用户的框，然后在3和4 位置输入修改后的密码，点击5位置的保存就修改完毕。

# 2  修改QB密码

2.1 用《Moviepilot使用手册》里提供的QB链接 和 账号密码登录上QB

![image-20260301173039538](/images/moviepilot修改密码教程_非公开/image-20260301173039538.png)

 在点击上图1位置后会弹出选项的框，然后点击2位置切到WebUI菜单在3位置输入修改后的密码，下拉到最下面点击下图4位置保存就修改完毕。

![图片](/images/moviepilot修改密码教程_非公开/image-20260301182634112.png)



2.2 修改MP和QB的关联密码

![image-20260301183241654](/images/moviepilot修改密码教程_非公开/image-20260301183241654.png)

在登录MP网页后依次点击上图1和2和3的位置后会弹出qb配置框，然后在4位置输入修改后的密码，点击5位置的保存 **还要点击6位置再次保存** 才修改完毕。



2.3 修改浏览器插件PTD和QB的关联密码

![image-20260301183721880](/images/moviepilot修改密码教程_非公开/image-20260301183721880.png)



在浏览器界面依次点击上图1和2和3的位置后会弹出编辑下载服务器框，然后在下图4位置输入修改后的密码，点击5位置的完成 修改完毕。

![image-20260301184042846](/images/moviepilot修改密码教程_非公开/image-20260301184042846.png)





# 3  修改EMBY密码

用《Moviepilot使用手册》里提供的emby链接 和 账号密码登录上emby

![image-20260301192544770](/images/moviepilot修改密码教程_非公开/image-20260301192544770.png)



在登录emby网页后依次点击上图1和2和3的位置后会弹出用户配置界面，然后点击下图4位置进入密码修改界面，然后在下图5和6 位置输入修改后的密码，点击7位置的保存密码， 修改完毕。

![image-20260301192949996](/images/moviepilot修改密码教程_非公开/image-20260301192949996.png)





# 4  修改jellfin密码

用《Moviepilot使用手册》里提供的jellyfin链接 和 账号密码登录上jellyfin

![image-20260301193932970](/images/moviepilot修改密码教程_非公开/image-20260301193932970.png)



在登录jellyfin网页后依次点击上左图1和2的位置后会弹出上右图的个人信息界面，然后在上右图的3位置输入当前密码，默认“admin@admin”，然后在上右图的4和5 位置输入修改后的密码，点击6位置的保存密码， 修改完毕。



# 5  修改飞牛影视密码

5.1登录飞牛后点击飞牛影视

![image-20260301194837630](/images/moviepilot修改密码教程_非公开/image-20260301194837630.png)



进入飞牛影视界面，依次点击上图1和2的位置后会弹出上图的账号-修改密码界面，然后在上图的3和4 位置输入修改后的密码，点击5位置的保存修改， 修改完毕。

5.2 修改MP和飞牛影视的关联密码

![image-20260301195547240](/images/moviepilot修改密码教程_非公开/image-20260301195547240.png)

在登录MP网页后依次点击上图1和2和3的位置后会弹出飞牛的配置框，然后在4位置输入修改后的密码，点击5位置的确认 **还要点击6位置再次保存** 才修改完毕。
